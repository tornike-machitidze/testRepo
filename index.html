Main Git Commands

  • git init: (ინიციალიზაციას ახდენს (რეპოსიტორის) არსებულ ფოლდერში)Initialize a Git repository in the current folder
  • git status: (აჩვენებს თუ რაიმე შეუნახავი ცვლილებები გვაქვს კოდში) Show the current status of your Git repository (the “working tree” status)
  • git add . : (ამატებს იმ შეუნახავ ცვლილებებს ყველას ერთად რომელსაც სტატუსი აჩვენებს) Track changes of all files in your Repository
  • git commit -m "your message":(აკომიტებს და ინახავს საბოლოო ვერსიას რაც ადდით შევინახეთ) Save updated code to a new Commit named “your message”
  • git checkout branch-name:(გადავყავართ იმ ბრენჩზე რომელსაც მივუთითებთ branch-name-აქ, როგორც ბრანჩის სახელს) Stands on branch-name branch
  • git checkout ID:(git log-ში გამოტანილი აიდი შეგვიძლია მივცეთ დადაგვაბრუნებს იმ კომიტზე, ეს არ ინახება უსაფრთხოა) Jump to a specific Commit of a Branch (commitid should be the ID of the Commit you want to checkout)
  • get reset --hard:(უსაფრთხო არაა ! შლის მის შემდეგ დამატებულ კომიტებს და გვაბრუნებს იმ ცომიტზე რომელსაც გადავცემთ ამასაც აიდის სახით, მას შემდეგ დამატებულ კომიტებს წაშლის) Turn the selected Commit into the new HEAD
  • git checkout -- .:(ბოლო შენახული კომიტის შემდეგ თუ იმუშავე ბევრი კოდი დაწერე მაგრამ არ მოგწონს და გინდა მარტივად გაასუფთაო ეს შეუნახავი დაუკომიტებებლი ცვლილება და დაბრუნდე ბოლო კომიტზე) Jump back to the last Commit and remove any untracked changes
  • git checkout -b branch-name:(ქმნის ახალ ბრანჩს, რომელშიც შეგიძლია იზოლირებულად იმუშაო ისე რომ მაინ ბრანჩზე არაფერი შეიცვლება. ახალი ბრანჩის შექმნა შეუძლია ამ კომანდსაც უბრალოდ ეს ქმნის და მასზე არ დგება git branch branch-name) Create a new Branch named branch-name
  • git merge branch-name:(როდესაც მორჩები ახალ ბრანჩში მუშაობას ის შეგიძლია ხილული გახადო მასტერ მაინ ბრანჩისთვის ამ კომანდით, დადგები მასტერზე და გადაცემ იმ ბრანჩის სახელს, რომელიც მასტერისთვის გინდა ხილული იყოს) Merge two Branches, branch-name is the Branch you merge with the Branch you’re currently working in
  • git branch -D branch-name:(შლის ბრანჩს მაგრამ თუ დამერჯილია ეს ცვლილებები არ წაიშლება ამისთვის კომიტის წაშლაა საჭირო) Delete the Branch named branch-name
  • git log: (აჩვენებს ყველა კომიტის აიდის და რომელია ბოლო ანუ ჰედ კომიტი მაგას)
  • git push: (თუ დაკლონილი რეპოზიტორზე მუშაობ ეს მასტერ ბრანჩზე რო დადგები დაფუშავს ცვლილებებს ანუ ატვირთავს remote repositoryze იმაზე რომელიც გიტჰუბზე)
  • git push origin -u branch-name: ( ეს იმ შემთხვევაში ცალკე ბრანჩზე მუშაობ და იქ ამატებ ცვლილებებს რემოუთზე)
